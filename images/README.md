# Images Folder

Add your images to this folder with the following filenames:

## Location Images (for Areas section)
- **jerusalem.jpg** - Jerusalem cityscape (recommended: 1200x800px)
- **tel-aviv.jpg** - Tel Aviv skyline or beach (recommended: 800x600px)
- **herzliya.jpg** - Herzliya beach/marina (recommended: 800x600px)
- **or-yehuda.jpg** - Or Yehuda cityscape (recommended: 800x600px)
- **rishon.jpg** - Rishon LeZion cityscape (recommended: 800x600px)

## Other Required Images
- **hero-bg.jpg** - Homepage hero background, Jerusalem skyline (recommended: 1920x1080px)
- **team.jpg** - Your team or personal photo (recommended: 600x800px)
- **favicon.png** - Browser tab icon (32x32 or 64x64px)
- **og-image.jpg** - Social media preview image (1200x630px)

## Optional Images
- **testimonial-1.jpg** - Client photo 1
- **testimonial-2.jpg** - Client photo 2
- **testimonial-3.jpg** - Client photo 3
- **blog-1.jpg** - Blog post image 1
- **blog-2.jpg** - Blog post image 2
- **blog-3.jpg** - Blog post image 3
- **cta-bg.jpg** - CTA section background

## Free Image Sources
- [Unsplash](https://unsplash.com) - Search "Jerusalem", "Tel Aviv", etc.
- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)

## Image Tips
- Use high-quality JPG format for photos
- Optimize images before uploading (use tools like TinyPNG or Squoosh)
- Keep file sizes under 500KB for faster loading
- Use landscape orientation for location images
